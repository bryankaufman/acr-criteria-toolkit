# ACR Appropriateness Criteria Toolkit

ACR Appropriateness Criteria automation including PDF downloader, parser, and database builder for clinical decision support.

## Features

- **PDF Downloader**: Automated ACR criteria downloads
- **Parser**: Extract clinical decision rules
- **Database Builder**: Structured storage for CDS
- **API**: Query interface for applications

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
# Download ACR criteria
python download_criteria.py

# Parse and build database
python build_database.py
```
